
# 🛍️ Catálogo Golosinas Aries

**Golosinas Aries** es un catálogo online pensado para facilitar la venta de productos de forma visual, rápida y sencilla.  
Permite a los clientes explorar artículos, agregarlos al carrito y realizar pedidos directamente por **WhatsApp**, sin necesidad de registro ni pagos en línea.

---

## ✨ Características principales

- 📱 **Diseño responsive** — se adapta perfectamente a celulares, tablets y computadoras.  
- 🛒 **Carrito de compras interactivo** — agrega, elimina o modifica productos sin recargar la página.  
- 💬 **Envío directo por WhatsApp** — genera el mensaje con los productos seleccionados para confirmar el pedido.  
- 🖼️ **Visualización clara de productos** — imágenes, precios y descripciones en una interfaz simple.  

---

## 🧩 Tecnologías utilizadas

- **HTML5 / CSS3 / JavaScript**
- **JSON / LocalStorage**
- **Git / GitHub Pages** (para despliegue)
- **VSC** (entorno de desarrollo)
---

## 🚀 Cómo usarlo

1. Abrí el catálogo desde el navegador.  
2. Explorá los productos disponibles.  
3. Agregá los que quieras al carrito.  
4. Confirmá tu pedido por WhatsApp con un clic.
